The Guds Must Be Fed

In Theme 4, CSS animations and Javascript were introduced as tools for creating interactive elements on websites. We were tasked with designing and developing a game using these tools, from concept to implementation. This involved creating custom graphics using Adobe Illustrator, converting them to SVG files, and incorporating them into the game using CSS and Javascript to add interactivity and animation. Through this process, we learned how coding can bring websites to life with engaging and interactive animations.

Overall, this course has provided valuable insights into the power of coding for web design and development.
